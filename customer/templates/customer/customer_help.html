Help for Customer Profile

Customer Profile is maintained in a database. Like every database table, CRUD ( Create, Read or View, Update and Delete) is neccessary. In this table, We decided to avoid DELETE but rather to make a profile active or inactive.

Create

Add is used to create a profile. Alternatively, profiles can be created from csv file by importing the file, viewing and saving or updating (if record already exist) to the database table.

View or Read multiple Profiles

It is important to view, see or display the profiles and that can be done through the categories. Click on any of the circled number to view the profiles accordingly.

View or Read single Profile

The detailed view is to read a single profile, which can be updated through the Update button at the buttom of the page

Update

Since update will require you to pick a profile, going through the view will be required before you can update a profile. The blue coloured Business name column is a link to the single View of the profile
